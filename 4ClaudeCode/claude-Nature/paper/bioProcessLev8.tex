\begin{abstract}
\noindent
Level 8 establishes an end-to-end reproducibility record for CausalBool analyses and introduces external validation anchors spanning (i) CRISPR dependency signals (DepMap integration pipeline) and (ii) paired tumor/normal corruption (paired-design analysis). We freeze canonical sign conventions and evaluation semantics for algorithmic efficiency and differential complexity, regenerate manuscript figures, and record deterministic checksums for the resulting artifacts. In the current repository state, DepMap integration is exercised against DepMap Public 24Q4 (Chronos) using the integrated \texttt{CRISPRGeneEffect.csv} matrix and a 10-node EGFR signaling scaffold; we derive a node-level essentiality proxy as the negative of the Chronos gene effect averaged across DepMap models for the mapped genes (so higher values indicate stronger dependency). At scaffold scale, the association between mean in-silico removal impact ($\Delta D$) and this dependency proxy is negative and not statistically resolved (Pearson $r=-0.41$, $p=0.245$, $n=10$ nodes). The paired corruption pilot uses a synthetic paired cohort (TCGA-BR; $n=100$ tumor/normal pairs) generated by scaffold perturbations, showing a consistent tumor simplification signal in $D^{(v2)}$ (mean $\Delta D^{(v2)}=-2.53$, paired $p=1.05\times 10^{-14}$).
\end{abstract}

\section*{Level 8 Definition Contract}
All Level 8 results adhere to a frozen contract recorded in \texttt{bitacora-lev8.md}. The canonical definitions used in the generated artifacts are:
\begin{itemize}
    \item Structural complexity proxy (GRN corpus): $D_{\mathrm{gzip}}(cm) = \mathrm{len}(\mathrm{gzip}(cm.\mathrm{tobytes}()))$ (compressed bytes).
    \item Structural complexity proxy (cancer/DepMap): $D^{(v2)}(cm)$ computed by the universal $D^{(v2)}$ encoder on the adjacency matrix (bits).
    \item Degree-preserved null: Maslov--Sneppen edge swaps with $n_{\mathrm{swaps}} = 20N$, seeded.
    \item Efficiency z-score: $z = (\mathbb{E}[D_{\mathrm{null}}] - D_{\mathrm{bio}})/\mathrm{sd}(D_{\mathrm{null}})$ (so $z>0$ indicates algorithmic efficiency).
    \item Node removal impact: $\Delta D(v) = D(G\setminus v) - D(G)$ (so $\Delta D>0$ means removal increases complexity; the node contributed to compressibility/efficiency).
    \item Corruption shift (paired TCGA): $\Delta D^{(v2)} = D^{(v2)}_{\mathrm{tumor}} - D^{(v2)}_{\mathrm{normal}}$ (so $\Delta D^{(v2)}<0$ indicates tumor structural simplification).
    \item Proxy boundary: all $D$ values are encoding-dependent proxies and are only compared within the same proxy/encoder and frozen serialization; no cross-proxy comparisons are interpreted quantitatively.
\end{itemize}

\section*{Data and Artifacts}

The Level 8 figure regeneration uses the processed GRN corpus under \path{data/bio/processed}, with summaries emitted to \path{paper/figures/results_summary.csv} and \path{paper/figures/supplementary_table_per_network.csv}.

\subsection*{DepMap external anchor (integration state)}
DepMap integration is run against DepMap Public 24Q4 stored at \path{data/depmap}. The primary dependency signal is the Chronos model-level gene-effect matrix \path{data/depmap/CRISPRGeneEffect.csv}, with metadata from \path{data/depmap/Model.csv}. For the EGFR scaffold validation, we derive a gene-level dependency proxy per mapped gene by averaging gene effect across all DepMap models (ignoring missing values) and negating it (so higher values indicate stronger dependency), then aggregating to nodes using the node-to-gene map. Raw file checksums and the exact run command are recorded in \texttt{bitacora-lev8.md}.

The current repository state includes a synthetic paired cohort under \path{data/cancer/patients/} containing 100 tumor/normal pairs (\texttt{TCGA-BR-XXXX\_Tumor.json}, \texttt{TCGA-BR-XXXX\_Normal.json}) generated by scaffold perturbations of a 10-node EGFR signaling model, with per-patient metadata at \path{data/cancer/clinical_metadata.csv}. \section*{Results}
\subsection*{Algorithmic efficiency (Figure 1)}
Regenerated outputs show a consistent positive shift in $z$ under the canonical sign convention, but with smaller effect sizes than earlier narrative drafts. Across the current run output table (\texttt{results\_summary.csv}, $n=231$ networks), the degree-preserved null comparison yields mean $z \approx 0.72$ and median $z \approx 0.53$, with approximately 21\% of networks achieving one-sided $p \le 0.05$ in the expected direction.

\subsection*{Essentiality prediction (Figure 2)}
The extended essentiality evaluation over \texttt{essentiality\_prediction\_dataset.csv} (642 genes, 20 networks) is currently diagnostic rather than confirmatory. Under the frozen $\Delta D(v)=D(G\setminus v)-D(G)$ definition and the present label sources, $\Delta D$ performance is near chance: bootstrap AUC $0.547$ (95\% CI $[0.442, 0.645]$) and 5-fold cross-validated AUC $0.461 \pm 0.080$; graph baselines (degree, betweenness) are similar or weaker.

\subsection*{DepMap pilot validation (Figure 3)}
DepMap integration is exercised on the synthetic TCGA-BR cohort using the 10-node EGFR signaling scaffold. For each tumor network, we compute node-wise removal impact $\Delta D(v)=D^{(v2)}(G\setminus v)-D^{(v2)}(G)$ and average across patients (100 tumor networks). DepMap Public 24Q4 supplies model-level gene effect values (negative = more essential); we define the dependency proxy used in this pipeline as the negative of the gene effect (so higher values indicate stronger dependency). Across the 10 scaffold nodes, the association between mean $\Delta D$ and dependency is negative but not statistically resolved at this scale (Pearson $r=-0.41$, $p=0.245$; Spearman $\rho=-0.48$, $p=0.162$; MI $=0.00$ bits).

\subsection*{Paired tumor/normal corruption (synthetic TCGA-BR; Figure 4)}
Using the synthetic paired TCGA-BR cohort, we compute within-patient corruption as $\Delta D^{(v2)} = D^{(v2)}_{\mathrm{tumor}} - D^{(v2)}_{\mathrm{normal}}$ on the adjacency matrix of each paired Boolean model. Across $n=100$ tumor/normal pairs, tumor networks show lower structural complexity than matched normal controls (mean $D^{(v2)}_{\mathrm{normal}}=46.52$, mean $D^{(v2)}_{\mathrm{tumor}}=43.99$, mean $\Delta D^{(v2)}=-2.53$), with a strong paired effect ($t=9.09$, $p=1.05\times 10^{-14}$). The coupling between $\Delta D^{(v2)}$ and survival is weak in this synthetic cohort (Pearson $r=0.13$, $p=0.205$).

\section*{Reproducibility and Regression Control}
Level 8 executes the Python and WolframKernel test suites and records the aggregate status in \texttt{bitacora-lev8.md}. All Level 8 manuscript artifacts (figures and validation outputs) are recorded with SHA-256 checksums.

\section*{Implications for Gate C}
Under the frozen decision thresholds (Table~\ref{tab:lev8_gate_thresholds}), Gate C is not satisfied in this checkout: the DepMap anchor is a small-scaffold pilot ($n=10$) and the essentiality incremental-value criterion does not clear the predefined margin beyond degree. The DepMap analysis pathway is implemented end-to-end but does not yet constitute external validation in the current repository state: the dependency table is a proxy artifact and the scaffold is small ($n=10$ nodes), limiting both statistical power and confound control. A Gate C pass requires rerunning the same pipeline against a provenance-complete DepMap release and reporting baselines/controls under a leakage-safe design.

The paired corruption pilot demonstrates a paired-design signal under $D^{(v2)}$ on a frozen scaffold, but the cohort is synthetic. Upgrading this pillar to Gate C requires replacing the synthetic pairing with a real paired acquisition route (TCGA tumor + matched normal), freezing preprocessing, and reproducing the paired shift with external anchors.
