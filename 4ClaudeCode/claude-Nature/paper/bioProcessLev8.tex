\begin{abstract}
\noindent
Level 8 establishes an end-to-end reproducibility record for CausalBool analyses and introduces external validation anchors using DepMap CRISPR gene effect data (24Q4) and paired TCGA tumor/normal RNA-seq cohorts. We freeze canonical sign conventions and evaluation semantics for algorithmic efficiency and differential complexity, regenerate manuscript figures with immutable checksums, and run a full automated test sweep across Python and WolframKernel suites. DepMap integration provides provenance-complete dataset acquisition and a pilot node-level validation on a 17-node oncogenic signaling scaffold by correlating mean in-silico knockout structural impact ($\Delta D_{\mathrm{KO}}$) with DepMap dependency (Pearson $r=-0.35$, $p=0.20$, $n=15$ nodes). Paired TCGA cohorts enable a within-patient tumor-vs-normal corruption analysis, showing a significant negative shift in tumor structural complexity under the frozen expression-to-network corruption protocol (mean $\Delta D^{(v2)}=-15.66$, paired $p=9.81\times 10^{-13}$; $n=50$ pairs).
\end{abstract}

\section*{Level 8 Definition Contract}
All Level 8 results adhere to a frozen contract recorded in \texttt{bitacora-lev8.md}. The canonical definitions used in the generated artifacts are:
\begin{itemize}
    \item Structural complexity proxy (GRN corpus): $D_{\mathrm{gzip}}(cm) = \mathrm{len}(\mathrm{gzip}(cm.\mathrm{tobytes}()))$ (compressed bytes).
    \item Structural complexity proxy (cancer/DepMap): $D^{(v2)}(cm)$ computed by the universal $D^{(v2)}$ encoder on the adjacency matrix (bits).
    \item Degree-preserved null: Maslov--Sneppen edge swaps with $n_{\mathrm{swaps}} = 20N$, seeded.
    \item Efficiency z-score: $z = (\mathbb{E}[D_{\mathrm{null}}] - D_{\mathrm{bio}})/\mathrm{sd}(D_{\mathrm{null}})$ (so $z>0$ indicates algorithmic efficiency).
    \item Knockout impact (DepMap validation): $\Delta D_{\mathrm{KO}}(v) = D(G) - D(G\setminus v)$ (so $\Delta D_{\mathrm{KO}}>0$ indicates structural impact under knockout).
    \item Corruption shift (paired TCGA): $\Delta D^{(v2)} = D^{(v2)}_{\mathrm{tumor}} - D^{(v2)}_{\mathrm{normal}}$ (so $\Delta D^{(v2)}<0$ indicates tumor structural simplification).
\end{itemize}

\section*{Data and Artifacts}

\subsection*{DepMap 24Q4 external anchor}
DepMap 24Q4 Public artifacts were acquired from the Figshare+ distribution endpoint and verified via SHA-256 checksums (see \texttt{bitacora-lev8.md}). A derived gene-level dependency table was constructed by averaging CRISPR gene effect across all cell lines:
\begin{equation}
\mathrm{Dependency}(g) = \frac{1}{|\mathcal{L}_g|}\sum_{\ell \in \mathcal{L}_g} \mathrm{GeneEffect}(\ell,g).
\end{equation}
This derived table is stored at \path{data/depmap/24Q4/derived/depmap_24Q4_gene_effect_mean.csv}.

\subsection*{Paired TCGA tumor/normal RNA-seq cohorts (pilot acquisition + pairing)}
To move beyond synthetic ``TCGA-BR'' pathway cohorts and enable within-patient tumor-vs-normal comparisons, we acquired real TCGA RNA-seq gene expression quantification from the NCI GDC API using the \texttt{STAR - Counts} workflow and explicitly enforced tumor/normal pairing by selecting GDC cases present in both \texttt{Primary Tumor} and \texttt{Solid Tissue Normal} queries. We froze a small paired pilot cohort of 5 tumor/normal pairs per project for the following set: TCGA-BRCA, TCGA-LUAD, TCGA-COAD, TCGA-PRAD, TCGA-KIRC, TCGA-HNSC, TCGA-THCA, TCGA-LUSC, TCGA-LIHC, TCGA-KIRP. Raw per-sample augmented STAR count tables and derived gene $\times$ sample count matrices are stored under \path{data/cancer/tcga_paired/<PROJECT>/raw/} and \path{data/cancer/tcga_paired/<PROJECT>/processed/}. Using the frozen expression-to-network corruption protocol recorded in \texttt{bitacora-lev8.md}, we instantiated paired patient-specific oncogenic signaling Boolean models under \path{data/cancer/patients_zanudo_prolif/<PROJECT>/}, with the paired cohort index at \path{data/cancer/tcga_index_zanudo_prolif_byproject.csv}.

\subsection*{DepMap pilot validation (Figure 3)}
DepMap 24Q4 integration enables a provenance-complete external validation anchor. We validate a 17-node oncogenic signaling scaffold (\path{ginsim_2020-mammal-oncogenic-signaling_2018_zanudo_proliferation}) by correlating per-node mean in-silico knockout impact $\Delta D_{\mathrm{KO}}$ (averaged across a $n=50$ TCGA tumor cohort) with DepMap dependency (gene effect mean; negative = more essential). Node $\rightarrow$ gene mapping is frozen and multi-gene nodes are aggregated by mean dependency. The association is negative but not statistically significant at $n=15$ DepMap-covered scaffold nodes (Pearson $r=-0.35$, $p=0.20$; Spearman $\rho=-0.31$, $p=0.27$; MI $=0.25$ bits).

\subsection*{Paired TCGA tumor/normal corruption (Figure 5)}
Using paired TCGA models instantiated on the same 17-node scaffold, we measure a within-patient corruption signal as $\Delta D^{(v2)} = D^{(v2)}_{\mathrm{tumor}} - D^{(v2)}_{\mathrm{normal}}$ computed on the adjacency matrix of each paired Boolean model. Across $n=50$ tumor/normal pairs, tumor models show lower structural complexity than matched normal controls (mean $D^{(v2)}_{\mathrm{normal}}=107.96$, mean $D^{(v2)}_{\mathrm{tumor}}=92.29$, mean $\Delta D^{(v2)}=-15.66$), with a strong paired effect ($t=-9.53$, $p=9.81\times 10^{-13}$). The corruption magnitude scales with inferred perturbation count (Pearson $r=-0.95$, $p=3.77\times 10^{-25}$), consistent with the frozen mechanistic scaffold in which constitutive perturbations sever regulatory inputs and simplify network structure.

\section*{Implications for Gate C}
The DepMap anchor provides the required external dataset provenance and a concrete analysis pathway, but it does not yet satisfy Gate C as a publishable biological punch because the external validation is node-limited: even after expanding scaffold coverage, the test operates at $n=15$ nodes, which constrains statistical power and makes confound control difficult. The next Gate C pass requires scaling node coverage (larger scaffolds and/or multi-scaffold aggregation) and adding context-matched controls (degree, expression proxies, copy-number) under a leakage-safe evaluation design.

Paired TCGA cohorts reduce substrate risk and provide a paired-design, real-data-dependent corruption signal within a frozen expression-to-network scaffold. The remaining Gate C work is to generalize beyond a single scaffold, quantify robustness across mapping/threshold choices at scale, and connect the corruption signal to external biology in a context-matched design (lineage-matched DepMap and clinically relevant endpoints).

The lineage-matched DepMap sensitivity sweep is direction-consistent with the context-agnostic anchor but does not change evidential status: at $n=15$ nodes, statistical power is still insufficient to treat the DepMap association as confirmatory. The multi-lineage sweep preserves negative direction across all tested lineages (Table~\ref{tab:depmap_lineage_sweep}), reinforcing that the primary bottleneck is model coverage rather than data provenance.
